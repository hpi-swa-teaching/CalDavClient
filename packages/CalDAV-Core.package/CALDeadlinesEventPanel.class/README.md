A CALDeadlinesEventPanel is a CALPanel for deadline events of a week in the DeadlinesWindow.  

Instance Variables
	app:			aCALApp 
	days:			aCollection 
	weekNumber:	aNumber 

app
	- the CALApp the DeadlinesEventPanel is shown in 

days
	- a Collection of days in the week 

weekNumber
	- number of the week in a year
