A CALMonthPanel is a CALPanel containing a month's events and buttons to switch between the months. 

Instance Variables
	app: 			aCALApp 
	days: 			aCollection 
	firstOfMonth:	aDate 

app
	- the CALApp the MonthPanel is placed in 

days
	- an OrderedCollection of CALDays

firstOfMonth
	- the first Date of the month
